⚙ Pengaturan Aplikasi

💾 Backup Data ke Komputer

Unduh seluruh data (siswa, jurnal, absensi, keaktifan, catatan kejadian, nilai, tujuan pembelajaran) sebagai file JSON. Simpan sebagai arsip atau cadangan sebelum menghapus data.

📂 Restore / Import Backup

Upload file backup JSON yang sebelumnya diunduh dari aplikasi ini. Data akan ditambahkan (bukan mengganti) data yang sudah ada.

🗑 Hapus Data per Kategori

Hapus data tertentu saja. Berguna untuk memulai semester baru — misalnya hapus data absensi dan nilai, tapi pertahankan daftar siswa dan tujuan pembelajaran.

⚠ Reset Seluruh Data

Hapus semua data transaksi (jurnal, absensi, keaktifan, kejadian, nilai, TP). Data siswa tidak dihapus. Tindakan ini tidak dapat dibatalkan — pastikan sudah backup terlebih dahulu.

🔑 Ganti Password Login

Password hanya tersimpan di kode aplikasi. Setelah diganti, unduh file HTML baru dan upload ulang ke GitHub.

Buka file index.html di text editor, cari baris const APP_PASSWORD = '...', ganti isinya, lalu upload ulang ke GitHub.
